Telco Asked Questions

A computer network is a collection of computers and other devices that are interconnected through communication media to share resources, such as files, printers, and internet connections.

LAN (Local Area Network): A network that covers a small geographic area such as home, work, or school.

WAN (Wide Area Network): A network that covers a large geographic area, such as between cities or countries, often using an internet connection or a dedicated line.

Router: A device that connects a local network to a wider network, such as the internet, and routes data traffic between those networks.

Switch: A device that connects multiple devices in a single local network (LAN) and manages the data transmitted between those devices.

An IP address is a numeric address used to identify devices in a network. There are two versions: IPv4 (example: 192.168.1.1) and IPv6 (example: 2001:0db8:85a3:0000:0000:8a2e:0370:7334).

An Ethernet cable is a type of cable used to connect devices to a local network (LAN) via an Ethernet port. These cables are usually Cat5e, Cat6, or Cat7.
